# RepoShoot

Space shooter game taking place in your own repository! Fly through your own source code and blast all those bugs!

## Usage

```
npx reposhoot
```

Run inside any repository

## Controls

- **Arrows / WASD** - Move
- **Space** - Fire
- **ESC** - Pause
